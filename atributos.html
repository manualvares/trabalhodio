Atributos

Alguna atributos que vão facilitar sua vida são:

  • id - indentifica no JS, qual elemento voce quer.
  • Style - pro CSS, é pra formatar da forma que quer.
  • class - pro CSS, serve para padronizar
  • src = "endereço de uma imagem" - serve pra colocar uma imagem
  • width"numero" - define a largura de uma imagem
